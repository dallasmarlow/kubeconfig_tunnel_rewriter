# kubeconfig_tunnel_rewriter

```
Usage of kubeconfig_tunnel_rewriter:
  -cluster-name string
    	cluster name to rewrite configuration for
  -kubeconfig string
    	kubeconfig file path
  -tunnel-addr string
    	tunnel IP address or hostname (default "127.0.0.1")
  -tunnel-port int
    	tunnel TCP port number (default 8443)
```
